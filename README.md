# Frontyc

Simple, powerfull and flexible front-end environment built with Gulp.

## Install

```bash
$ npm install frontyc -g
```

## Use

In your terminal:

```bash
$ ftyc init
$ npm i
$ ftyc
```

### Browser Support

Targeting evergreen browsers and **IE10+**.
